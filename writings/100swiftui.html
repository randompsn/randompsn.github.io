I finished the coding challenge. I was so out of my mind that I have to copy other's solutions to deduce my error.

                
enum SquareRootErrors: Error {
  case Out
  case None
}

func SquareRoot(_ number:Int = 1, end: Int = 10_000) throws -> Int {
    if (number < 1 || number > 10_000) {
        throw SquareRootErrors.Out 
    }
    
    for i in 1...100 { 
        if i * i == number {
            return i
        }
    }
    
    throw SquareRootErrors.None
}


do {
  let result = try SquareRoot(16)
  print("The squareroot of this number is \(result).")
} catch SquareRootErrors.Out{
  print("Oops, can't do numbers outside of 1 to 10000.")
} catch SquareRootErrors.None{
  print("This number doesn't have a integer square root!")
} catch {
  print("There was an error.")
}

Some errors I made: 1. Not using "in" in "for i in 1...100" 2. within the do element, I wrote SquareRootError without s at the end. The latter took me about 10 minutes to deduce. I want to sleep now.
